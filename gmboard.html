📋 Table Editor

All tables are editable from the Tables section. Changes are saved automatically to localStorage and take effect immediately. The ↺ Restore button resets everything to original values.

The type field in the Event Table uses fixed keywords: encounter · loot · camp_nemico · other · nothing.

The type field in the Complication Table uses: Encounter · Environment Damage · Environment · None (case sensitive).
